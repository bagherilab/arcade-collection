get_cell_state(volume: float, critical_volume: float, threshold_fractions: dict[str, float]) → str[source]#

Estimates cell state based on cell volume.

The threshold fractions dictionary defines the monotonic thresholds between different cell states. For a given volume v, critical volume V, and states X1, X2, …, XN with corresponding, monotonic threshold fractions f1, f2, …, fN, a cell is assigned state Xi such that [f(i - 1) * V] <= v < [fi * V]. Cells with v < f1 * V are assigned state X1. Cells with v > fN * V are assigned state XN.

Parameters:

  • volume – Current cell volume.

  • critical_volume – Critical cell volume.

  • threshold_fractions – Critical volume fractions defining threshold between states.

Returns:

Cell state.
